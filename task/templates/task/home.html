Welcome to Your Task Manager !!!

What you can do here :

  • Create a Task.
  • Update your present Task.
  • Change the Status of your Task.
  • Delete a Task when completed.

For Developers

  • A Neat representation of all the APIs using Swagger is available.
  • All APIs follow RESTfull principles.
  • Easy Access, Integration and secure APIs.

Tough Security Principles

  • Credentials of the User are stored following most secure methods.
  • Passwords are Hashed using PBKDF2 algorithm with a SHA256 hash.

Developers Security

  • Used a most Authentic and Secure method for our Database security.
  • Followed JWT Authentication, DRF Token Authenctication for validating the usage of APIs
  • Token Changing for every stipulated time frame, makes a more Rigid layer to security.

Who can do What

Guest User
  • User logIn
  • User Register.
  • Create tasks.
  • Update tasks.
  • View all the tasks of the user.
  • Detail view of a particular Task of that user.
  • Delete their own task.
Admin
  • Can See all the task irrespective of the user.
  • Can delete a Task if found any illegal terms.
  • Can edit/update a Task, removing any illegal terms.
SuperUser
  • Can Create Admin Users
  • Can See all the task irrespective of the user.
  • Can delete a Task if found any illegal terms.
  • Can edit/update a Task, removing any illegal terms.

Developers APIs

  • Register User
  • LogIn User
  • Get Token for User
  • Get User Profile
  • Update User Profile
  • Get all the Tasks of User
  • Get detail of the Task of User
  • Update a Task of User
  • Delete a Task of User
  • Change the status of the Task of a User

Summary :

Tech Stack Used : Python, Django, Django REST framework, HTML, CSS, JavaScript, Bootstrap, SQLite

Security Methods Used : Hashing by PBKDF2 algorithm with a SHA256 hash, JWT Authentication, DRF Token Authenctication

Implemented 3 types of users, and assigning some basic rules to them.

Made RESTfull APIs, in prespective of Developers.
